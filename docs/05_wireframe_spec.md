## WF-09: AI 톡 채팅 화면

> 연결 기능: MVP-01 AI 톡 — 일상 대화 + 컨텍스트 수집

### 목적

대화 흐름 속에서 AI 전화영어로의 전환을 유도하는 중심 허브 역할을 한다.
푸시 알림 단발성이 아닌, 지속적 관계형 인터페이스로 수신 거절률을 낮추고 학습 습관을 형성한다.

### 주요 컴포넌트

| 영역 | 컴포넌트 | 설명 |
|---|---|---|
| 채팅 영역 | AI 메시지 (한/영 혼용) | 한국어 대화 속에 자연스럽게 영어 표현 삽입 |
| 채팅 영역 | 사용자 메시지 | 한국어 또는 영어 자유 입력 |
| 채팅 영역 | 표현 코칭 카드 (Phase 2 추가 예정) | 대화 맥락에 맞는 영어 표현 1~3개 추천 |
| 채팅 영역 | 전화 전환 유도 메시지 | "이따 전화로 연습해볼까?" + 인라인 응답 버튼 |
| 입력 영역 | 텍스트 입력 필드 + 전송 버튼 | 표준 메시징 UI |

### 사용자 액션
- **메시지 입력 + 전송**: AI와 자유 대화 (한국어/영어 모두 가능)
- **[좋아!] 탭**: AI 전화 예약 확인 → 예약 시간 토스트 표시 + 전화 대기 상태 진입
- **[시간 변경] 탭**: 시간 피커 등장 → 원하는 시간 선택 후 예약
- **[오늘은 패스] 탭**: 전화 건너뜀 → "괜찮아요! 내일 다시 물어볼게요 💪" 메시지 표시
- **입력 중 한국어 감지**: Phase 2에서 WF-10 표현 코칭 UI 추가 예정

### 개선 포인트
- **Before**: 10분 전 푸시 알림 1회 → 단발성, 맥락 없음, 40% 수신 거절
- **After**: 하루 종일 대화 관계 유지 → 전화가 대화의 자연스러운 연장선
- **핵심 가치**: 알림이 아닌 대화 — 사용자가 이미 AI와 관계를 형성한 상태에서 전화를 받으므로 심리적 장벽 대폭 감소

## WF-10: 톡 내 표현 코칭 UI (Phase 2)

실시간 영어 표현 코칭을 제공하여 자연스러운 학습 기회를 만든다.

### 주요 컴포넌트

| 영역 | 컴포넌트 | 설명 |
|---|---|---|
| 코칭 카드 | 한글→영어 변환 | 사용자 한국어 입력을 casual/professional 두 가지 톤으로 번역 |
| 코칭 카드 | 영어 교정 | 사용자 영어 입력의 문법/표현 오류 감지 및 교정 제안 |
| 코칭 카드 | 설명 텍스트 | 왜 이 표현이 더 좋은지 한 줄 설명 |
| 액션 버튼 | [복사] / [이걸로 보내기] | 코칭 표현을 클립보드에 복사하거나 바로 메시지로 전송 |
| 액션 버튼 | [교정해서 보내기] | 교정된 영어 표현으로 메시지 대체 전송 |
| 닫기 버튼 | ✕ | 코칭 카드 닫기 (강제가 아닌 선택적) |

### 사용자 액션
- **한국어 입력 시**: 자동으로 영어 변환 코칭 카드 표시 (0.5초 디바운스)
- **영어 입력 시**: 오류 감지 시 교정 코칭 카드 표시
- **[이걸로 보내기] 탭**: 선택한 영어 표현으로 메시지 전송
- **[복사] 탭**: 클립보드에 복사 (사용자가 직접 수정 후 전송 가능)
- **[교정해서 보내기] 탭**: 교정된 표현으로 메시지 대체 전송
- **[닫기] 탭**: 코칭 카드 닫힘 (원래 한국어/영어 그대로 전송 가능)

### 개선 포인트
- **학습 임베딩**: 별도 학습 세션이 아닌, 일상 대화 중 자연스러운 학습 기회
- **두 가지 톤**: 같은 의미를 casual/professional로 비교 → 상황별 표현력 향상
- **비강제적 설계**: 코칭 카드는 선택적 — 무시하고 한국어로 보내도 됨
- **교정 학습**: 영어 입력 시 실시간 피드백 → 스픽의 발음 교정에 대응하는 텍스트 표현 교정

## WF-01: 톡 내 전화 직전 브리핑

> 연결 기능: MVP-02 AI 톡 → 전화 자연 전환 (Ready 단계)

| 브리핑 카드 | 핵심 표현 3개 | 통화에서 사용할 표현 미리보기 |
| 액션 | [자세히 보기] 링크 | 탭 시 WF-02 브리핑 확장 카드 펼침 |
| 액션 | 인라인 버튼 3개 | 준비됐어! / 5분 뒤 전화 / 오늘은 건너뛸게요 |
| 푸시 알림 | 축약 알림 배너 | 앱 밖에서도 알림 수신 → 탭 시 AI 톡 화면으로 이동 |

### 사용자 액션
- **[준비됐어!] 탭**: 전화 대기 상태 → "좋아요! 곧 전화할게요 💪" 응답 메시지 + 1~3초 내 전화 발신
- **[5분 뒤 전화] 탭**: 발신 5분 지연 → "알겠어요! 5분 뒤 다시 알려줄게요 👍" 응답 메시지
- **[오늘은 건너뛸게요] 탭**: 오늘 전화 취소 → "괜찮아요! 내일 다시 준비해둘게요" + 내일 예약 미니 카드
- **[자세히 보기] 탭**: 브리핑 카드 확장 → WF-02 확장 카드 표시
- **푸시 알림 탭 (앱 밖)**: 앱 실행 → AI 톡 화면으로 이동, 브리핑 메시지 위치로 자동 스크롤

### 개선 포인트
- **Before**: 독립 푸시 알림 → 알림 센터에서 소비되고 끝, 맥락 단절
- **After**: 톡 대화 흐름 속 브리핑 → 하루 대화의 자연스러운 마무리로 전화 전환
- **대화 맥락 연속**: 오전에 "분기 보고 있어요"라고 대화했다면, 브리핑이 그 맥락을 이어받음
- **이중 알림**: 톡 내 메시지 + 푸시 알림 동시 발송 → 앱 안팎 모두 커버

- **스크롤 구조**: 섹션별로 자연스럽게 스크롤 → 각 섹션이 독립적으로 소화 가능

| 진입 화면 | 추천 시나리오 카드 | 레벨에 맞는 시나리오 1개 자동 추천 (소요 시간 + 난이도 명시) |
| 진입 화면 | 실전 감각 메시지 | 롤플레이의 즉각적 가치를 사용자 언어로 전달 |
| 진입 화면 | Primary CTA | "지금 바로 시작하기 (3분)" — 소요 시간 명시로 부담 최소화 |
| 체험 화면 | 체험 모드 배너 | "힌트 제공 ON" 명시 → 첫 체험자 안심 |
| 체험 화면 | 3분 타이머 | 짧은 시간 → 압박 없는 체험 가능 |
| AHA Moment | 완료 결과 + 발화 통계 | 작은 성취 시각화 → 긍정 감정 형성 |
| AHA Moment | 7일 챌린지 진행 바 | 1/3 완료 표시 → "2번만 더 하면 된다" 동기 부여 |

### 사용자 액션
- **[지금 바로 시작하기] 탭**: 체험 모드 롤플레잉 시작
- **[다른 시나리오 보기] 탭**: 전체 시나리오 카탈로그 화면으로 이동 (선택 후 돌아와 시작)
- **[건너뛰고 홈으로] 탭**: 온보딩 건너뜀 → 홈 화면 (단, 추후 "롤플레잉 아직 안 해보셨어요!" 배너 노출)
- **AHA Moment [무료 체험 계속하기] 탭**: 홈 화면으로 이동 (7일 챌린지 시작)
- **AHA Moment [지금 결제하기] 탭**: 결제 화면으로 이동

### 개선 포인트
- **Before**: 앱 설치 → 레벨테스트 → 홈 화면 (기능 많고 뭐부터 해야 할지 모름)
- **After**: 레벨테스트 → 3분 롤플레잉 → AHA Moment → 결제 CTA (명확한 단일 경로)
- **"3분"의 심리적 효과**: 짧은 시간 명시로 "일단 해보자"는 결심 유도
- **건너뛰기 버튼 위치**: 작게 하단에 → 시각적으로 덜 두드러지게 (기본 경로 강화)
- **실전 감각 메시지**: 내부 전환율 데이터 대신 사용자가 공감할 수 있는 가치 메시지로 참여 유도

## 디자인 가이드 노트

### 색상 방향
- **Primary CTA**: 링글 시그니처 그린 계열 (수신/시작/준비됐어 버튼)
- **거절/건너뛰기 버튼**: 회색 / 텍스트 링크 형태 (시각적 우선순위 낮게)
- **Grow 리포트**: 따뜻한 톤 (성취·칭찬 맥락) — 차갑고 딱딱한 수치 UI 지양
- **AI 톡 채팅**: 부드러운 배경색 + 대화 말풍선 구분
  - AI 메시지: 밝은 회색 배경, 좌측 정렬
  - 사용자 메시지: 링글 그린 계열 배경, 우측 정렬
  - 브리핑 카드: 강조 배경색 (연한 블루/그린 그라데이션) — 일반 대화와 시각적 구분

### 타이포그래피 방향
- 주요 액션 텍스트: 굵게 + 명확하게 (무엇을 해야 하는지 즉시 파악)
- 피드백/설명 텍스트: 작고 부드러운 서체 (읽을 부담 없이)
- 영어 표현: 영어 전용 폰트로 가독성 확보
- **AI 톡 메시지**: 자연스러운 메시지 톤 — 교과서적 딱딱함 지양, 친구와 대화하는 느낌

### 인터랙션 방향
- **카운트다운 타이머**: 숫자 변화 애니메이션 (긴박감 없이 부드럽게)
- **스트릭 +1**: 불꽃 아이콘 튀어오르는 애니메이션 (작은 기쁨)
- **바텀 시트**: 슬라이드업 애니메이션 (표준 iOS/Android 패턴)
- **힌트 카드 펼침**: 아코디언 펼침 (부드러운 높이 변화)
- **톡 메시지 도착**: 부드러운 슬라이드인 + 미세한 바운스 (메시징 앱 표준)
- **(Phase 2) 표현 코칭 카드 등장**: 입력 필드 위로 페이드인 + 슬라이드업 (0.3초)
- **브리핑 카드 확장/접힘**: 아코디언 펼침 — 톡 화면 내에서 자연스럽게 확장

### 채팅 인터페이스 가이드라인
- **메시지 간격**: 시간 간격 30분 이상일 경우 타임스탬프 구분선 표시
- **브리핑 구분선**: "📞 전화 브리핑" 라벨과 함께 시각적 구분선으로 일반 대화와 분리
- **인라인 버튼**: 메시지 카드 내부에 인라인 액션 버튼 배치 (외부 팝업 최소화)
- **(Phase 2) 톤 전환 탭**: 입력 영역 바로 위, 항상 노출 (탭 전환 시 부드러운 색상 변화)
- **(Phase 2) 표현 코칭 카드**: 입력 필드와 톤 탭 사이에 위치, 닫기 버튼 항상 표시 (강제 아님)
- **타이핑 인디케이터**: AI 응답 생성 중 "..." 애니메이션 (메시징 앱 표준)
- **스크롤 동작**: 새 메시지 도착 시 자동 스크롤 (사용자가 위로 스크롤 중이면 중단)
