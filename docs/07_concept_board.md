## 1. 컨셉보드 목표 및 전략

### 목적
- 링글 블로그 공개 → '좋아요' 반응 극대화 → 확산 점수 10% 획득
- 심사가 아닌 **감성 공감 + 브랜드 신뢰 + 공유 욕구** 자극
- RING LOOP 솔루션 컨셉을 한 장에 강렬하게 전달

### 핵심 전략
```
복잡한 논리 설명 X → 직장인이 즉각 공감하는 감성 언어 O
기능 나열 X → 단 하나의 강렬한 비전 이미지 O
링글 내부 자료 느낌 X → 세련된 매거진/광고 크리에이티브 O
```

### 타겟 독자
- 링글 블로그 방문자 (기존 유저 + 영어 학습 관심자)
- 직장인 (28~40세, 비즈니스 영어 니즈)
- SNS 공유 가능성이 높은 공감형 콘텐츠 소비자

## 2. 핵심 슬로건 후보 + 최종 결정

### 슬로건 후보 A (LCP 기능 중심)
> **"오늘 하루를 말해보세요. 링글이 오늘 영어를 만들어드릴게요."**

- 분석: Daily Log 입력 → Context Card 생성 플로우를 직접적으로 전달. "말해보세요"가 참여를 유도
- 강점: LCP의 핵심 UX를 직관적으로 설명, 행동 유도(CTA) 성격
- 아쉬운 점: 두 문장이 길어 임팩트가 분산됨. 블로그 썸네일 한 눈에 각인 어려움

---

### 슬로건 후보 B ★ **최종 선택**
> **"일상이 곧 스크립트. 링글 AI가 오늘을 영어로 바꿉니다."**

- 분석: "일상이 곧 스크립트"는 LCP의 핵심 가치(일상 → 학습 재료 변환)를 5자로 압축. "오늘을 영어로 바꿉니다"는 즉각적 변환 능력 강조
- 강점: ① 짧고 강렬해 블로그/SNS 썸네일에서 즉각 각인 ② LCP 기능 함의 최강 ③ 심사위원이 솔루션 개념을 슬로건 한 줄에 파악 가능 ④ "스크립트"라는 단어가 영어 학습자에게 친숙하면서도 신선함
- 부제 연결: "RING LOOP — 준비(Ready) → 통화(In) → 성장(Grow)"

---

### 슬로건 후보 C (감성 공감형, 서브 카피 후보)
> **"영어가 두렵지 않았으면. 링글이 먼저 준비할게요."**

- 분석: 직장인의 심리적 장벽(영어 공포)을 정면으로 겨냥. 공감도 최상
- 강점: 블로그 좋아요 반응 유도력 강함. 감성 파트너십 이미지
- 아쉬운 점: LCP의 "일상→학습 재료" 개념이 전혀 반영되지 않음. 기존 RING LOOP 슬로건과 차별화 부족
- **서브 카피 활용**: 컨셉보드 하단 부제 또는 우측 비주얼 영역 보조 텍스트로 활용 권장

---

### 슬로건 최종 결정: **후보 B**
> **"일상이 곧 스크립트. 링글 AI가 오늘을 영어로 바꿉니다."**

**선택 이유 (공모전 심사 + 블로그 확산 점수 최적):**
1. **기억에 남음**: "일상이 곧 스크립트" 5자가 강렬한 임팩트. 읽는 즉시 핵심 개념 각인
2. **LCP 기능 함의 최강**: 일상 → 스크립트(Roleplay Script/Phone Brief) 변환이 한 줄에 담김. 심사위원이 솔루션 개념을 즉시 파악 가능
3. **감성적 공감**: "오늘을 영어로 바꿉니다"는 직장인의 일상 변화 욕구에 직결
4. **확산 적합**: 짧고 임팩트 있어 SNS 공유·블로그 썸네일에서 경쟁력 높음

**서브 카피 (컨셉보드 하단):**
- 메인 서브: "매일 톡 한 마디, 링글 AI가 오늘의 영어를 준비합니다"
- 감성 서브 (선택): "영어가 두렵지 않았으면. 링글이 먼저 준비할게요."

## 3. 레이아웃 구성안 (A4 가로, 297×210mm)

### 기본 그리드 (3분할 세로 구조)

```
┌────────────────────────────────────────────────────────────────┐
│  ← 전체 폭 297mm →                                              │

│ │    (좌측 35%)        │ │          (우측 65%)               │ │
│ │                      │ │                                   │ │
│ │ ① 메인 슬로건       │ │  스마트폰 화면을 보며              │ │
│ │   (대형 텍스트)      │ │  준비하는 직장인 이미지            │ │
│ │                      │ │  + RING LOOP 아이콘 오버레이       │ │
│ │ ② 서비스명           │ │                                   │ │
│ │   Ringle AI Speaking │ │                                   │ │
│ │                      │ │                                   │ │
│ │ ③ RING LOOP 3단계   │ │                                   │ │
│ │   Ready·In·Grow     │ │                                   │ │
│ │                      │ │                                   │ │
│ │ ④ 핵심 수치 1~2개   │ │                                   │ │
│ │   (하단)             │ │                                   │ │
│ │                      │ │                                   │ │
│ └──────────────────────┘ └───────────────────────────────────┘ │
│                                                                  │
│  [하단 띠] Ringle 로고 + 부제 카피 1줄 + (선택) QR코드          │
└────────────────────────────────────────────────────────────────┘
```

### 각 구역 상세 배치

#### 좌측 텍스트 영역 (35%)
- **상단 (10mm 여백 후)**: 부제 태그 "제5회 링글 서비스 기획 공모전" (소형, 회색)
- **중상단**: 메인 슬로건 (2줄, 대형 — 40~48pt)
  ```
  "일상이 곧 스크립트.
  링글 AI가 오늘을 영어로 바꿉니다."
  ```
- **중단**: 서비스명 레이블 "Ringle AI Speaking · RING LOOP" (미디엄, 브랜드 블루)
- **중하단**: LCP 플로우 아이콘 다이어그램 (수평 배치, 아이콘 + 짧은 텍스트)
  ```
  [💬 AI 톡] → [⚡ AI 변환] → [🎯 3채널 학습]
   톡으로 안부/근황       Topic Card           AI 튜터
                         Phone Brief      AI 전화(RING LOOP)
                         Roleplay Script      1:1 튜터
  ```
  → RING LOOP 원형 루프 (Ready·In·Grow)는 "AI 전화" 채널 아이콘 우측 하단에 소형 배치
- **하단**: 핵심 수치 카드 3개 (강조 박스)
  ```
  ┌──────────────────┐  ┌──────────────────┐
  │ 수신 거절         │  │ 결제 전환율      │
  │ 40% → 25%        │  │ 미사용 대비      │
  │ 로 낮춥니다       │  │ 최대 2.4배       │
  └──────────────────┘  └──────────────────┘
  (기존 3번째 카드 "톡 한 마디 → 3채널 학습 재료 자동생성"은 우측 비주얼에 통합)
  ```

#### 우측 비주얼 영역 (65%)
- **메인 이미지**: 직장인이 스마트폰을 들고 AI 전화를 준비/수신하는 장면
- **오버레이 그래픽**: RING LOOP 원형 아이콘 (반투명 오버레이로 이미지 위에 배치)
- **배경 그라데이션**: 깊은 네이비(좌) → 링글 블루(우) 그라데이션

#### 하단 띠 (전폭, 약 20mm 높이)
- 링글 공식 로고 (좌)
- 부제 카피: "AI가 준비하고, 함께 전화하고, 성장을 기억합니다" (중앙)
- QR코드 또는 앱 스토어 배지 (우) — 선택 항목

## 4. 비주얼 방향

### 이미지 컨셉: 하이브리드 (실사 + 그래픽 오버레이) 추천

**실사 이미지 (배경)**
- 주인공: 세련된 직장인 (30대 초반, 남성 또는 여성)
- 장소: 퇴근 후 카페 또는 집 소파 (피크 시간 21~23시 반영)
- 포즈: 스마트폰 화면을 집중해서 보거나, 이어폰을 착용하고 대화 중
- 표정: 자신감 있고 집중된 표정 (불안/당황 없음 — RING LOOP가 준비시킨 상태)
- 조명: 따뜻하고 밝은 톤 (저녁 조명, 프리미엄 느낌)

**그래픽 오버레이**
- **LCP 플로우 다이어그램** (이미지 상단 가로 띠): "AI 톡 대화 → 세 채널 카드" 변환 과정을 간단한 아이콘 플로우로 표현
  - [💬] → [⚡] → [🎯] 3단 아이콘 + 연결 화살표 (반투명 배경 패널 위)
- **RING LOOP 원형 루프 아이콘**: LCP 다이어그램 우측 하단 "AI 전화 채널" 안에 작게 배치 (Ready/In/Grow 3단계 원형)
- 색상: Ready(파란색), In(초록색), Grow(주황색) — 각 단계 색상으로 구분
- AI 톡 채팅 화면 UI 미니 목업: 스마트폰 화면 내에 AI 톡 대화 인터페이스 (카카오톡 스타일 채팅 + 실시간 표현 코칭 말풍선) 작게 표시

**스타일 레퍼런스**
- 링글 현재 광고 크리에이티브 톤 유지 (깔끔하고 지적인 느낌)
- Apple 광고의 "사람 + 기기 조합" 구성에서 영감
- B2C SaaS 랜딩 페이지의 히어로 이미지 스타일

## 6. 톤앤매너 가이드

### 브랜드 색상 방향
```
Primary Blue:    #1B5BFF (링글 브랜드 블루 추정)
Deep Navy:       #1A2B4B (텍스트, 배경 딥 컬러)
Ready 단계:      #4A9EFF (밝은 블루)
In 단계:         #34C77B (성장 그린)
Grow 단계:       #FF8C42 (에너지 주황)
Background:      #FFFFFF / #F5F7FF (화이트/연한 블루 배경)
```

### 폰트 방향
- 슬로건: Pretendard ExtraBold 또는 산돌 고딕 Bold (한국어 최적)
- 서비스명/RING LOOP: 영문 Helvetica Neue Bold 또는 Inter Bold
- 수치 강조: 숫자 특대형 (60~72pt), 단위/설명 소형 (14~16pt)
- 부제 카피: Pretendard Regular (Light 계열)

### 전반적 톤
- **지적이고 세련된** — 일하는 직장인이 신뢰할 수 있는 프리미엄 서비스
- **따뜻하고 공감적** — "링글이 당신 옆에 있다"는 파트너십 감성
- **자신감 있는** — 과장 없이 명확한 수치와 컨셉으로 설득
- **미니멀하되 임팩트 있는** — 공백을 두려워하지 않는 고급 레이아웃
