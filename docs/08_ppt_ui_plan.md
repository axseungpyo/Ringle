### 1.2 타이포그래피

```
헤딩 폰트:    Pretendard Variable (800 ExtraBold)
본문 폰트:    Pretendard Variable (400 Regular ~ 600 SemiBold)
코드/수치:    JetBrains Mono 또는 Pretendard Variable (900 Black, tabular-nums)

슬라이드 제목:    40~52px (700~800)
섹션 라벨:        18px (700, 대문자, letter-spacing: 2px)
본문:             24~30px (400~500)
캡션/출처:        16~18px (400, muted)
핵심 수치:        64~120px (900, Black)
```

### 1.3 공통 UI 요소

| 요소 | 스타일 |
|------|--------|
| Ringle 로고 | 우측 상단 (top: 36px, right: 48px, width: 120px), 일반 슬라이드 opacity 0.9, 타이틀/엔드 슬라이드 opacity 0.7 |
| 프로그레스 바 | 하단 5px, 그라데이션 (#5B4FFF → #7C3AED) |
| 슬라이드 카운터 | 우측 하단, 22px, rgba(0,0,0,0.3) |
| 태그 | border-radius: 24px, bg: #EDE5FF, color: #5B4FFF |
| 인사이트 바 | 네이비 배경 rounded-16px, 좌측 퍼플 배지 + 우측 화이트 텍스트 |
| 카드 | border-radius: 20px, border-left: 5px solid [색상별], 배경 rgba(255,255,255,0.7) |
| 섹션 구분 | 라벨 (대문자 + letter-spacing) + 제목 H2 조합 |

### 1.4 빌드 이펙트 전략

```
- 타이틀 슬라이드: step 1~5 (순차 등장)
- 데이터 슬라이드: step 1(라벨+제목), step 2(메인 콘텐츠), step 3(인사이트 바)
- 비교 슬라이드: step 1(제목), step 2(Before), step 3(After)
- 카드 나열: stagger 클래스 + --i 변수로 연쇄 등장
- 통계 슬라이드: 각 수치별 개별 step
```

### 1.6 데이터 근거 상세 (수치의 신뢰성 확보)

> **심사위원 관점**: 모든 수치에는 출처와 추론 과정이 명시되어야 한다.
> "~라고 추정" 대신 "~에 근거하여 ~로 산출" 형태로 표현.

| 핵심 수치 | 값 | 출처 | 신뢰 등급 | 사용 슬라이드 |
|----------|-----|------|---------|-------------|
| AI 전화 수신 거절률 | 40% | 링글 내부 데이터 (공모전 제공) | A (1차 데이터) | #7, #1, #20 |
| AI 발신 vs 사용자 발신 | 77.7% : 22.3% | 링글 내부 데이터 (공모전 제공) | A | #7 |
| 롤플레이 결제 전환율 | 미사용자 대비 2.4배 | 링글 내부 리서치 (공모전 제공) | A | #8, #17, #20 |
| 글로벌 AI 교육 CAGR | 32.2% | MetaTech Insights (2024) | B (외부 리서치) | #2 |
| 국내 AI 영어 MAU | 82만→156만→200만+ | Sensor Tower / Mobile Index (2024-25) | B | #2 |
| 영어교육 앱 신규 설치 | 85% 증가 | Mobile Index (2024.01 기준) | B | #2 |
| AI 콜센터 거부감 | 73.6% | 아시아경제 (2024) — "AI 전화 응대 만족도 조사" | B | #7 |
| 맥락 제공 시 거부감 감소 | 73.6%→42% | 위 조사 내 "사전 안내 제공 시 수용률" 항목 | B→C (외부→추론) | #20 |
| 직장인 영어=비즈니스 핵심 | 74.4% | 스픽 블로그 리서치 (2024) | B | #5 |
| SOM 시장 규모 | 3,000~5,000억 원 | Business Research Insights 기반 자체 추산 | C (추론) | #3 |
| 수신 거절률 목표 | 40%→25% | 맥락 제공 시 거부감 감소 비율(~40%) 적용 추산 | C | #20 |
| 온보딩 경험률 목표 | 70~80% | Duolingo 온보딩 완료율 벤치마크 (~70%) | C | #20 |

**근거 표기 원칙:**
- **A등급** (1차 데이터): 수치만 표기, 출처 작게 명시
- **B등급** (외부 리서치): "출처: [기관명] (연도)" 형태로 슬라이드 하단에 명시
- **C등급** (추론): "~에 근거한 추산" 형태로 명시, 보수적/낙관적 범위 함께 표기
